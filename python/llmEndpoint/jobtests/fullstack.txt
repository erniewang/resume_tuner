Essential Functions:

All functions are done as part of a coordinated team. This role will report to North Star management, but primary responsibilities and day-to-day tasks will come from the lead full stack developer. Primary responsibilities will include front and back-end development, database management, and tasks to enhance our current offerings.


Frontend Development

Develop and maintain user-friendly web interfaces using HTML, CSS, and JavaScript.
Ensure seamless navigation and responsiveness across various devices and browsers.


Backend Development

Design and implement server-side logic using PHP for robust and scalable web applications.
Develop and maintain databases (SQL) to ensure data integrity and efficient data retrieval.
Implement APIs for seamless communication between the front-end and back-end systems.


Database Management

Design and optimize database schemas for performance and scalability.
Write complex SQL queries for data retrieval, manipulation, and management.
Implement and maintain database security measures to safeguard sensitive information.
Maintain source code using SQL, PHP, JavaScript and various APIs.
Develop documentation on dashboard maintenance


Testing and Debugging

Conduct thorough testing of web applications to identify and fix bugs, ensuring optimal performance.
Debug and troubleshoot issues reported by users, providing timely resolutions.
Communication
Ensures that web enhancements and future additions are communicated appropriately with the firm.


Continuous Learning

Stay updated on industry trends and emerging technologies to enhance and broaden skill sets.
Proactively suggest and implement improvements to current development processes.


This description covers the primary and principal duties of the job. It is not intended to give all the details or a step-by-step account of the way each procedure or task is performed.
